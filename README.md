# polish-parliament-sentiment
This was my final project for Sentiment Analysis Class in which I participated at WNE UW. Its main purpose was to classify sentiment of Polish Parliament during 8th, 9th and partly 10th term. 
This project consists of 3 notebooks: the first one utilizes Sejm API (https://api.sejm.gov.pl) to download pertaining information (MPs, proceedings lists, stenograms), the second one is used to initially preprocess the data and the final one analyses and classifies the sentiment.
